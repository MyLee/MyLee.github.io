Vad jag har gjort

Jag bygger en Windows klassik spel "Mine sweeper" med javascript, html och css som man kan spela direkt i website. Det är optimal för datorskärm, det ser lite konstig på annan typ av skärmen. Med den här spelet kan användare lära sig strategi, theori och konsten av slum.

Hur det går

Det består av massor lucka då jag använder js för att skappa rutorna med id. Det är lättare för mig att använda vanlig tabel inte canvas. Vad är svåraste med uppgiften är att bygga rätt loop som ska öppna tomma luckorna och visa nummer av bomb. Det tar ganska mycket tid för 'debug' också dvs se till att spelet fungera som det ska.

Om man har tid

Om jag har med tid tänker jag lägga mer funktioner:

  • Visa tiden som användare använd för att lösa gåtan
  • Visa hur många bomb kvar
  • Olika svårighets nivå t.ex lätt, svårt, mycket svårt genom storlek och bomb mängde

  • Spara lokal, och på server för att jämför med varandra, visa bäst tid rekord att lösa en gåta
